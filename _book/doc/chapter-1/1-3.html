源代码和示例

源代码和示例

本书是开源的。你可以在 Github 中找到 Markdown 的源文件。本书会从社区接收到频繁的更新，因此需要确保检查 Github 仓库来获得最新的版本。

同时在一个伴随的仓库中拥有大多数示例的完整实现。查看 README 获取安装细节。

我们将采用 shapeless 2.3.2，Typelevel Scala 2.11.8+ 或 Lightbend Scala 2.11.9+ / 2.12.1+。
